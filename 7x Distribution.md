Q: How do I use the shop?  
A: Every message you send is the equivalent of 0.5 points. The more active you are, the more levels you progress, each level earning double points than the previous one (with a points cooldown of 1 message per 3 minutes, aka only one message you send after 3 minutes counts towards your points.) run (7)/balance to check your points

Q: Can I customize commands in NEO?  
A: NEO offers basic manual customization, meaning you must edit the code to edit behavior. For quick and simple command customization, consider Pro.

Q: What's DynamicGPT?  
A: DynamicGPT is a Pro-Only feature that uses AI to detect whether a query needs the complex GPT-4o Mode, GPT-4o Mini,l or the simple & quick GPT-3.5 Model, effectively merging   
both into one, allowing a conversation to switch between models effectively.
